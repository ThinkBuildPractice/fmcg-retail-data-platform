# FMCG Retail Data Platform

Production-inspired retail data engineering project built using PySpark and distributed ETL workflows.

## Project Overview

This project simulates enterprise-scale retail/FMCG data processing workflows inspired by modern cloud data engineering practices.

The platform processes retail transactional data to generate business-ready analytical datasets for operational monitoring and customer insights.

---

## Architecture

### Bronze Layer
Raw data ingestion

### Silver Layer
Data cleansing and validation

### Gold Layer
Business-ready analytical datasets

---

## Key Features

- Retail data ingestion pipeline
- Distributed PySpark transformations
- Data quality validation framework
- Customer satisfaction analytics
- Order performance monitoring
- SQL-based reporting
