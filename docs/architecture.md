# Retail Data Platform Architecture

## Data Flow

### 1. Ingestion Layer
Raw retail dataset ingestion

### 2. Bronze Layer
Raw storage

### 3. Silver Layer
Data cleansing and standardization

### 4. Data Quality Layer
Validation and anomaly detection

### 5. Gold Layer
Business analytical datasets

### 6. Analytics Serving Layer
SQL-based KPI reporting

## Workflow Orchestration

Managed using Apache Airflow DAG scheduling.

## Processing Engine

Distributed PySpark transformations.
